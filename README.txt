Basic Calculator Program

Create a simple Python program that asks the user to input two numbers and a mathematical operation (addition, subtraction, multiplication, or division).
Perform the operation based on the user's input and print the result.
Example: If a user inputs 10, 5, and +, your program should display 10 + 5 = 15.


PS: The code that has been written allows functionalities as per the given instructions which are:
		1. / (Division)
		2. * (Multiplication)
		3. + (Addition)
		4: - (Subtraction)
	Note: Division by Zero is not allowed.

Any operations that will be run aside from the above listed will initaite an error.
